# Mage Essentials

A Ragnarok Online classic style spell pack for Unreal Engine 5.7.
Drop-in: 11 sprite sheets + DataTable + C++ runtime (cast pipeline, projectile,
Fire Wall actor) + Niagara recipe + integration guide.

Skills included (levels 1-10 each):
- **Fire Bolt** — N bolts at level N, element Fire
- **Cold Bolt** — N bolts at level N, element Water
- **Lightning Bolt** — N bolts at level N, element Wind
- **Fire Wall** — 3-cell flame barrier; hits-per-cell scales with level

Read **UE5_INTEGRATION_GUIDE.md** for full setup instructions.
