## Key Decisions
- **Shared armature**: Use `base_f` and `base_m` models (naked, bald) for all animations. Individual class models pair their meshes to the shared skeleton. Adding a new class = ~5 min body atlas render.
- **Holdout occlusion**: Replaces depth_front arrays for ALL equipment. Equipment always renders in front, body holdout hides equipment behind body. Simpler and more reliable.
- **Always_front depth mode**: Standard for all equipment layers (weapons, headgear, shields, garments). `depth_map.json` still generated but ignored.
- **Hair as tintable layer**: Same pipeline as headgear but with material TintColor multiply for runtime recoloring. 9 canonical RO hair colors.
- **Reset-derive-reconcile**: Pattern for hair hiding — reset flag before equipment loop, set during loop, reconcile after. Used by both local and remote paths.
