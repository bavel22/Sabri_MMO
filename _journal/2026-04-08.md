## What Got Done

### Minimap texture not rendering — root cause & fix

**Symptom**: Minimap widget drew the frame, zone name, dots, and zoom buttons correctly, but the SceneCapture2D render target no longer showed the overhead scene.

**Root cause**: `PostProcessSubsystem::CreateCutoutMaterial` (added after the map system was committed, for the sprite-vs-world occlusion rework) pushes a post-process material into the unbound global `APostProcessVolume`. That material reads `PPI_CustomStencil` and darkens every pixel where stencil != 1 via `lerp(InColor, InColor * 0.35, 0.6)` → `InColor * 0.61`.

The minimap's `USceneCaptureComponent2D` uses `CaptureSource = SCS_FinalColorLDR`, which runs the full post-process chain on the capture — including the cutout. The minimap camera looks straight down from +5000 Z, so the player-sprite billboard (which faces the main camera) is edge-on and writes no stencil into the capture. The cutout then darkens the entire captured frame, which visually blends into the dark minimap frame background and reads as "the texture isn't showing".

**Fix**: Added `CaptureComponent->ShowFlags.SetPostProcessMaterial(false)` in `MinimapSubsystem::SetupOverheadCapture()` alongside the existing `SetFog/SetBloom/SetMotionBlur` disables. This excludes the cutout (and any other PP material) from the minimap capture while keeping tonemapping, exposure, and color grading intact. Editor target rebuilt cleanly — 3.76s incremental.

## Key Decisions

- Chose `ShowFlags.SetPostProcessMaterial(false)` over changing `CaptureSource` to `SCS_SceneColorHDR`. Reason: keeps tonemapping and color grading for a readable, exposure-correct minimap image. `SCS_SceneColorHDR` would give raw HDR linear which looks washed out or overly dark depending on scene brightness.
- Did not modify the cutout material itself. The cutout is working correctly for the main camera — the problem is purely that SceneCapture2D blindly inherits the global PP volume. Narrowly scoping the fix to the capture component keeps the cutout's behavior unchanged for all other cameras.
