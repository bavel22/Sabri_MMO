## Key Decisions
- **Slope rock = gameplay indicator** — 45-degree cutoff, rock texture = unclimbable terrain
- **Always version materials** — never overwrite, always create _01, _02, etc.
- **RO textures for decals, SDXL for sprites** — AI decals look garish, RO originals are perfect
- **Grass Output node is manual** — Python can't configure it, must be done in Material Editor
- **Painted vs Random grass** — two methods, both multiply with slope mask, can mix

## Prompts That Worked Well

**Texture generation** (ComfyUI):
> Describe COLORS not plants. "grass" generates individual blades. Use "warm golden-green earth tones" not "grass field". SeamlessTile + CFG 6.5 + 60 steps + LoRA 0.15

**Scatter sprite generation** (SDXL Base):
> Use "product render, centered, isolated on white background" — NOT Illustrious-XL. White-to-alpha threshold 200, bottom-align mesh pivot.

**Material iteration**:
> Always create numbered variants (_01, _02, _03) via Python script. Never overwrite. Reference `Material_Variant_Tracker.md` for current version.
