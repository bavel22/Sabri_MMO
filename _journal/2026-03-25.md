## Key Decisions
- Remove duplicate animations for the same animation — no more 2 idles, etc.
- Dual-pass render from the SAME .blend scene is the ONLY approach that guarantees pixel-perfect weapon overlay alignment. All separate-render approaches fail.
- Per-frame depth ordering computed in Blender at render time (not runtime heuristics).
- Weapon atlas `character` field must be `weapon_{view_sprite_id}` matching DB `items.view_sprite` column.
- Always delete old `.uasset` files before reimporting — UE5 does NOT auto-update from changed source files.
